## 🌐 Production Deployments & Full-Stack Applications

### 🪐 knoYOsta (Know Your Star)
* **Live Deployment:** [🔗 Live Web Application Link](https://amar-sreedhar-p.onrender.com)
* **Code Archive:** [📂 Old Project Archive](./Old_Project_Archive/)
* **Author:** Amar Sreedhar P

knoYOsta is a full-stack AI-powered astrology web application built with Flask, MongoDB, and Groq (Llama-3.3-70B), featuring secure authentication, RESTful APIs, and low-latency real-time inference.

#### 🔍 Engineering Highlights & System Design
* **⚡ Ultra-Fast Inference Pipeline:** Re-architected backend frameworks to mitigate API rate limits and latency bottlenecks. Integrated the **Groq API (Llama-3.3-70B)** to achieve near-instantaneous, low-latency text generation.
* **⚙️ Backend Ecosystem:** Developed robust RESTful APIs utilizing **Python & Flask** featuring strict structured request handling, secure routing, and configured CORS policies.
* **🗄️ Database & Security Infrastructure:** Engineered a scalable NoSQL storage layer via **MongoDB Atlas & PyMongo**. Enforced secure credential storage using **bcrypt-based password hashing** and decoupled sensitive keys using strict environment variables (`.env`).
* **💻 Frontend Interface:** Designed a highly responsive UI with **HTML5 and CSS3**, backed by asynchronous **JavaScript (ES6+)** for non-blocking API calls and dynamic DOM updates.
* **☁️ Cloud Deployment:** Deployed and maintained backend microservices end-to-end via **Render & AWS** infrastructure.
