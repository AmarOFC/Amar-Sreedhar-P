# knoYOsta (Know Your Star)
&emsp;
knoYOsta is a full-stack AI-powered astrology web application built with Flask, MongoDB, and Groq (Llama-3.3-70B), featuring secure authentication, RESTful APIs, and low-latency real-time inference.
